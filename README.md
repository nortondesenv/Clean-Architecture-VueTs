# Wine Vue

Projeto [demo](https://main.d30rzvge0k24d.amplifyapp.com/) .

#### Aplicação composta

-   Vue + TypeScript
-   Sass/Scss
-   Jest
-   Clean Architecture
-   BEM

#### Instruções

Instalar dependências:

```sh
$ npm install
```

Compila ambiente de desenvolvimento:

```sh
$ npm run serve
```

Roda os testes da aplicação:

```sh
$ npm run test
```

Correções lint:

```sh
$ npm run lint
```
